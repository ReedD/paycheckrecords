# Paycheck Records

A CLI prompt to quickly enter in hours at paycheckrecords.com.

Their web interface is too tedious and slow.

---

## Installation

1. Clone this repo
2. ```npm install```
3. Create and edit ```config.js```
4. ```npm start```
5. Enter the date and hours in the prompt
