# Paycheck Records

Automatically fill in today's hours at paycheckrecords.com.

------------------

## Installation

###### Clone this repo

```
git@bitbucket.org:thriveline/paycheckrecords.git
```

##### Install dependencies

```
npm install
```

###### Edit the settings

```
subl index.js
```

###### Run the script

```
npm start
```
